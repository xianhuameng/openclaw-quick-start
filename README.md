## 核心功能
### 本地文件管理神器
- 核心能力：智能分类、快速检索、批量处理本地文件
- 效率提升：**81%**（对比传统文件管理方式）
- 核心场景：文档整理、素材归档、重复文件清理

### 个人知识库
- 一键存档：网页、学术论文、GitHub 仓库内容自动抓取并结构化存储
- 智能检索：支持关键词、语义检索，快速定位知识库内容
- 多格式兼容：支持 Markdown、PDF、Word 等主流格式

### 日程管理
- 核心亮点：**微信截图秒变日历事件**（自动识别时间、地点、待办事项）
- 多端同步：支持与系统日历、飞书日历、钉钉日历同步
- 提醒功能：自定义时间提醒，避免遗漏重要事项

### 自动化工作流
- 基础能力：定时任务、网站状态监控、AI 日报自动生成
- 配置方式：可视化配置/代码配置，新手友好
- 示例场景：每日 9 点自动生成昨日工作日报并发送至企微群

### Skills 扩展
- 内置技能库：**1715 个技能**，覆盖办公、创作、数据分析等多场景
- 自定义技能：支持根据个人需求开发/扩展专属技能
- 技能调用：简单指令即可触发组合技能

### 多平台集成
一键接入主流办公沟通工具，无需切换多端：
- 飞书
- 企业微信
- 钉钉
- QQ

## 高级功能
### 高级配置
针对专业用户的深度优化能力：
- 多模型切换：支持切换不同 AI 模型（如 OpenAI、智谱、文心一言等）
- 成本优化：模型调用量监控、按需切换低成本模型
- 性能调优：本地缓存、并发控制、资源占用优化

### 高级自动化工作流

- 触发条件：支持多条件触发（如「当网站监控异常 + 工作时间 → 立即推送告警」）

## 辅助指南
### 常见问题速查
| 问题类型 | 核心问题 | 解决方案 |
|----------|----------|----------|
| 部署问题 | 启动失败/端口占用 | 检查端口是否被占用，重启服务并核对部署指令 |
| 功能问题 | 微信截图无法识别 | 确保截图包含清晰的时间/事件信息，更新 OCR 插件 |
| 性能问题 | 检索速度慢 | 开启本地缓存，优化知识库索引 |

### 避坑指南（新手必看）
1. 部署前：提前检查本地环境（如 Python 版本、Docker 状态），避免版本不兼容
2. 使用中：不要一次性导入超大知识库（建议分批次），避免卡顿
3. 扩展时：自定义 Skills 先测试再上线，避免影响核心功能
4. 安全上：本地部署注意权限设置，避免文件/数据泄露
